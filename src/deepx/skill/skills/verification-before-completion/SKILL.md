# Verification Before Completion

## 执行检查

完成每个功能后，执行以下检查：

1. **功能完整性** — 功能描述中的每一点都实现了吗？
2. **边界情况** — 空输入、超长输入、特殊字符都处理了吗？
3. **错误处理** — 异常情况有兜底吗？
4. **代码质量** — 有没有明显的 lint 问题？
5. **测试覆盖** — 核心逻辑有测试吗？
6. **无遗留问题** — 文件都保存了吗？临时的 print/log 都清理了吗？
